# Fullstack Portfolio

A personal portfolio website for presenting my web development projects in one place. The site highlights interactive front-end work, small JavaScript games, backend practice projects, and internship assignments while keeping each project easy to open, play, and review.

## Overview

This repository is built as a project showcase for Sumit Singh. The main page links to selected projects with live demos and source folders, making it useful as both a portfolio site and a record of full-stack learning progress.

## Featured Projects

- **Simon Game**: A browser-based memory game where players repeat an increasing color and sound sequence.
- **Virtual Drum Kit**: An interactive drum kit that responds to keyboard presses and clicks with drum sounds.
- **Dice Game**: A simple two-player dice roll game that randomly selects a winner.

## Tech Stack

- HTML5

- JavaScript
- Node.js practice projects
- React/Vite practice work

## How to Run

Open `index.html` directly in a browser.

For the best local preview, you can also serve the folder with a simple static server:

```bash
npx serve .
```

Then open the local URL shown in the terminal.
